# Why the Web UI is the product

## One sentence

**breve is different because you open a browser and a continuous 3D multi-agent world is already running — then you can talk to invent the next one.**

## Not this

- Not “clone and fight autotools from 2007”
- Not “install Unity to see three balls fall”
- Not “ask ChatGPT for code and debug imports for an hour”
- Not “another README that says multi-agent and shows a terminal print”

## This

1. `breve-web` → localhost  
2. Canvas is **alive** (autoplay gravity / mass)  
3. One-click curriculum  
4. Optional natural language → **safe JSON scene** → same live view  
5. **Share** so teaching and demos spread  

The Python engine, physics, and classic `Control` / `Mobile` API make that honest and extensible.  
The **Web UI is why strangers care in the first thirty seconds.**

## Pitch for GitHub / talks

> NetLogo is 2D. Unity is heavy. LLM paste is dead code.  
> **breve** is a living 3D multi-agent sandbox in the browser — with an optional AI door.
