# Notices & credits

## Original breve

**The breve Simulation Environment** was created by **Jonathan Klein**
(Hampshire College and subsequent work), roughly 2000–2009, with bitrot
fixes through ~2015.

- Upstream repository: https://github.com/jonklein/breve
- Historical site: http://www.spiderland.org/
- License: GNU General Public License v2 (see `GPL.txt` / `LICENSE`)

This project is a **community revival** of that work. It is **not** an
official product of the original author unless stated otherwise by them.

Please preserve attribution when you fork, package, or publish derivatives.

## Academic citation

If you use breve in research or teaching materials, please cite the original
work, for example:

> Klein, J. (2002). *BREVE: a 3D environment for the simulation of
> decentralized systems and artificial life.* Proceedings of the Eighth
> International Conference on Artificial Life (Artificial Life VIII).

See also `CITATION.cff`.

## This revival

Modern Python 3 engine, browser UI, AI scene builder, and packaging added
by revival contributors (2025–). Legacy C++ / steve / Python 2 sources are
kept under `legacy/` for historical reference.

## Third-party

- Optional xAI / Grok API for natural-language scene building (user-supplied key)
- three.js (CDN) for the browser viewport
- NumPy, moderngl, FastAPI, and other dependencies — see `pyproject.toml`
